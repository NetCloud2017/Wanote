# 常用正则表达式

国内手机号码：

使用的时候要看看当前的号码有没有新出的。现在有 17 开头的了。

```javascript
/^(13[0-9]|14[57]|15[012356789]|18[012356789])\d{8}$/g;
```

邮箱：

```javascript
w+([-+.]w+)*@w+([-.]w+)*.w+([-.]w+)*
```

每隔三个数字加一个逗号:

```JavaScript

'123123211312.333333'.replace(/(?=(?!^)(?:\d{3})+(?:\.|$))(\d{3}(\.\d+$)?)/g, ',$1');
// 123,123,211,312.333333

'-1019234801211.2142345678'.replace(/(?<!._\..\*)(\d)(?=(\d{3})+($|\.))/g, '$1,');
// 简单些的
let reg = /(?!^)(?=(\d{3})+\b)/g
```
