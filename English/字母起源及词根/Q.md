# Q

## 字母起源

> 腓尼基字⺟ Q — 猴⼦ — 尾巴 — 排队

queue  [kju:] 队伍 原意是尾巴 辫⼦

> ⾳变 qu = c

quarter [ˈkwɔ:tə(r)] 四分之⼀， cross ⼗字路⼝

quake [kweɪk] 震动

queen [kwi:n] 王后， 来⾃古英语 cwen ，c 的⾳变，皇后的帽⼦和 cap 同源

> qua 词根 = c ，和 can 同源，can 能 — 能⼒ — 知识 — 探索

question ['kwestʃən] 问题，quest 词根

qualify [ˈkwɒlɪfaɪ] 质量，qual 性质，ify 动词
