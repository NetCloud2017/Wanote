## vim 模式简介

vim 的模式三种分别是 normal mode、visual mode、 command mode， 在初始启动是 normal mode，在 normal mode 下, 进入 visual mode 可以按 <b> v </b> 键， 进入 command mode 可以按 <b> shift + : </b>; 退出可以按 <b> Esc </b> 键或者按 <b> j j </b> 键两下。 推出后就是 normal mode。
