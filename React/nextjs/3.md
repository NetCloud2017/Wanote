# nextjs 路由

## layout 布局 和 root layout (根布局)

布局会保留原来的状态，根布局必须要有html 和 body 标签， 可以通过分组路由创建多个根布局。

## template

最后输出是布局包裹模板， 模板包裹page, 模板不会维持状态，每次切换路由都会为children重新创建新的实例

## 实现路由导航的4种方式

1. `<Link>` 组件


2. `useRouter()` hook , 注意，它要被用到客户端组件才行

```js

import {useRouter} from 'next/navigation'

const router = useRouter()

router.push('/app')

```

3. `redirect` 函数

```js
import {redirect} from 'next/navigation'

redirect('/login')

```

4. history API 

window.history.pushState 和 window.history.replaceState
