## 型变

型变(variant)即类型改变，也可以理解是类型的变通，TS 在编
译期间，不同类型的值是能够赋值的。

```TypeScript

let a: number = 2
let b: string = 'join'

type Man = {
    name: string;
}
type Person = {
    name: string;
    age: number;
}

let man: Man = {
    name: 'John',
}
const person: Person = {
    name: 'John',
    age: 18
}

man = person;
```

了解型变，需要了解父子类。主要有两种：

1. 对于索引类型，子类是父类的超集；

2. 对于联合类型，子类是父类的子集。

   子类 child 是父类 parent 的子集。但 child 同样比 parenti 更具体。举例：选择题 ABCD 排除掉两项后，答案所在的范围就更具体了（比如 AB)；或者 1 extends number,.1 相对于 number 更具体，number 又类似于 012…所以联合类型中子类是父类子集，且更具体。
